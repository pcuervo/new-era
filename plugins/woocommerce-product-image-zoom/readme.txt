WooCommerce Product Image Zoom - user can see every aspect of your product with this zoom.

== Description ==

WooCommerce Product Image Zoom - Nice zoom that works with both PC and mobile and show user image as is.

= Features: =

* Easy to use
* Variety of options for zoom
* Mobile-friendly




= Next Release ( 1.1.0 ) =

= Features: =
* zoom level with mouse scroll
* option to enable/disable changing zoom on scroll

= Changes: =
* Optimize and then Refactor the code
* Updating and fixes to options. Not all of them are working yet


== Installation ==

1. Install WooCommerce Zoom Image either via the WordPress.org plugin directory, or by uploading the files to your server
2. Activating WooCommerce AJAX Filters.
3. In Admin go to WooCommerce Settings and disable "Enable Lightbox" option.
4. That's all.
